## Overview
Once you done with the WebSocket request saving, you or your team might need to implement that code into your codebase. To save the time and make this process fast Firecamp introduce the code-snippet feature.

Just click on the `Code` button as shown in the image, You can have a code-snippet modal screen. you can select the message from the collection to generate ready to use `code-snippet` for specific language or library.
